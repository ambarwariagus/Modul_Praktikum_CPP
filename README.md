# Modul Praktikum C++

Repositori ini berisi modul-modul praktikum pemrograman C++ yang dirancang untuk mahasiswa, praktisi, maupun siapa pun yang ingin memperkuat dasar-dasar pemrograman C++ secara sistematis. Modul-modul ini secara khusus disusun sebagai fondasi menuju pemrograman High Performance Computing (HPC) dan komputasi paralel.

Modul ini mencakup berbagai topik inti seperti:
- Struktur data dasar  
- Object-Oriented Programming (OOP)  
- Struktur data modern (`std::vector`)  
- File I/O  
- Pendahuluan konsep komputasi paralel  
- dan akan terus dikembangkan lagi 🚀

---

## 📌 Tujuan Repositori

Repositori ini dibuat untuk membantu:
- Menyimpan modul praktikum C++ secara tersusun dan terstruktur.
- Menyediakan sumber latihan dan referensi pembelajaran praktis.
- Menjadi acuan bagi pembelajaran C++ yang siap dipakai di kelas atau project pribadi.
